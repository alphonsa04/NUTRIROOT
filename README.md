# NutriRoot Project

## Overview
NutriRoot is a web application designed to provide smart farming insights and analytics. This project includes various functionalities such as user authentication, data visualization, and user management.

## Project Structure
The project consists of the following files:

- **index.html**: The main entry point for the application, containing the overall structure and links to other pages.

- **dashboard.html**: The user dashboard displaying relevant information and features for users after they log in.
- **register.html**: A registration form for new users to create an account.
- **forgot.html**: A page allowing users to reset their password if they forget it.
- **css/style.css**: Styles for the application, defining the visual appearance of the HTML pages.
- **js/google-auth.js**: Handles Google authentication logic, including managing user sign-in and token retrieval.
- **js/script.js**: Contains the main JavaScript logic for the application, handling user interactions and dynamic content.
- **.gitignore**: Specifies files and directories that should be ignored by Git version control.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to view the application.
4. Ensure you have a local server running if you are using features that require server-side processing.

## Usage Guidelines
- Users can register for a new account via the `register.html` page.
- Existing users can log in through the `login.html` page using their email and password or Google Sign-In.
- After logging in, users will be redirected to the `dashboard.html`, where they can view their smart farming insights.
- If users forget their password, they can reset it using the `forgot.html` page.
